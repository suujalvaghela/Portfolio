KPI-Pulse: Clean Architecture, Real-Time Systems

A production-grade multi-tenant SaaS backend built to demonstrate how clean architecture scales. Not a throwaway side project — a real system with real problems solved.

Problem

Small startups need affordable, real-time KPI tracking. Existing solutions are either expensive or over-engineered.

Solution

A clean, modular backend that's easy to understand and modify. No clever tricks, no premature optimization, just solid systems.

Secondary Featured Project

Instagram Brand Intelligence System

Automated D2C brand data extraction and validation. Built as a backend automation platform with real-time webhook bot and intelligent scraping pipeline.

Problem

D2C brands hide real websites behind Instagram profiles and aggregator pages. Verifying legitimacy and collecting social links requires manual investigation.

Solution

A Directus extension that listens to Instagram DMs, scrapes profiles, resolves links, validates websites, stores records, and replies with results.

Key Features

  • Real-time Instagram webhook bot for instant DM responses.
  • Aggregator-aware link resolution for Linktree-style pages.
  • Duplicate prevention to avoid re-scraping and API quota waste.
  • Social link collection for Facebook, LinkedIn, YouTube, TikTok, and X.
  • REST API support for batch processing.

Architecture Flow

DM → Webhook → Scraper → Validator → DB → Reply
